Portfolio

Web/Software

I’m constantly building new projects to learn everything from UI, DevOps, and RESTful APIs.

Culture-Stack

Culture Stack is a social media app that allows users to share their creative works and get feedback from other artists. We have created a platform for photographers to upload, categorize, and describe any challenges they are having with a particular process.

This app was built on a MERN Stack - using MongoDB, Express, ReactJS, and NodeJS. The app is hosted on Heroku, which does not allow local file storage. Because of this, I've set up cloud storage via AWS S3 for all user uploaded content.

NYT-Search

This was a simple app built to allow the user to search articles from the New York Times and save their favorite articles. This could be useful

NYT Search was also built on a MERN Stack - using MongoDB, Express, ReactJS, and NodeJS. It used the New York Times API to grab the articles based on the user's criteria.

MeHungry

MeHungry is a small project that I worked on with a partner in late 2017. The idea was to build an app that suggests local restaraunts, cafés, and bars to the user based on mood and preference. JJ worked on the front-end design while I worked on the API calls and javascript logic.

As silly as the idea seems, I learned a ton about the Google Maps, Google Places API, and javascript callbacks. We used Firebase to store previous seach parameters, allowing the user to recall previous search results the next time they visit the page.
